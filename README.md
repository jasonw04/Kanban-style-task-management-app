reasoning behind choices

design choices
-overall style and theme
    -same as assignment 1 we used https://www.canadacomputers.com/en/ as a basis as the inspiration of the colour theme, but we used wrike https://www.wrike.com/features/kanban-boards/ and the in class example as an inspiration for both layout and theme
    -uses a large header with buttons to reset and add a new board and has a a board and task counter with a label counter under it
    -uses larges cards to separate tasks and show visibility and hover effect for buttons to show clickability
    -different colour buttons to distinguish uses and labels to distinguish the type of task
    -tasks are separated into cards for each level of progress the task is in and has a drag and drop function that lets users put tasks whenever they need
    -shows the date of each task and when it is due with a label based on when it is due and the progress of it being done

colour palette
-using bright colour for important details like importance and level of completion - on track, due today and so on
-white is used to brighten up the view for the user so it is something the user can easily look at for things they need to do to associate it with importance and happiness
-associates tasks with something bright and easy to look at

insipration
-layout and styling was insipred by wrike and a little bit of canaada computers
